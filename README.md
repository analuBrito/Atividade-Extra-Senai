# 🌐 Portfólio de Ana Brito

Este projeto foi desenvolvido como parte da atividade prática extra do curso **Introdução à Programação Front-End e Back-End (IP 20h A)**.  
O objetivo é criar uma landing page funcional que apresente meu portfólio pessoal, reunindo informações sobre minha formação, habilidades e projetos desenvolvidos.  

---

## 📖 Introdução
Santa Catarina é um dos polos nacionais de crescimento em tecnologia, e a construção de um portfólio digital é essencial para estudantes em formação.  
Este portfólio serve como uma vitrine dos meus conhecimentos em **HTML, CSS e JavaScript**, além de demonstrar boas práticas de desenvolvimento e organização de código.  

---

## 🚀 Tecnologias utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Font Awesome  
- Google Fonts  

---

## ✨ Funcionalidades
- Alternar entre tema claro e escuro 🌙  
- Animações ao rolar a página   

---

## 📂 Estrutura do Portfólio
A página possui as seguintes seções obrigatórias:
1. **Cabeçalho**: nome, foto e breve descrição.  
2. **Sobre mim**: apresentação pessoal e trajetória.  
3. **Habilidades**: lista de competências técnicas e comportamentais.  
4. **Projetos**: lista de projetos com descrição, tecnologias e links.  
5. **Rodapé**: informações de contato e redes sociais.
